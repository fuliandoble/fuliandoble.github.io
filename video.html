Homecoming

I created an informal, documentary style video on my birth-searching journey and reunion in April 2025. The finished result is now available to watch on YouTube.

The name, 回家, was chosen because it translates to literally mean 'to return home' or to 'come home' and I was struck by how this was used by my birth family instead of the word 'reunion'

The editing was done by me using Adobe Premiere, filmed primarily using the Dji Osmo Pocket 3. I also added the translation with the help of Rujin Wang and our Changning translator 小白.

As this project is very personal, it is currently only set to unlisted.
